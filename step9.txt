---

# 🔌 API Endpoints

## 1️⃣ Home

GET `/`
→ API running status

---

## 2️⃣ Manual Prediction

POST `/predict`

Input:

```json
{
  "humidity": 70,
  "pressure": 1012
}
```

Output:

```json
{
  "temperature": 32.15
}
```

---

## 3️⃣ Weather Data

GET `/weather?city=mumbai`

→ Real weather data fetch karta hai

---

## 4️⃣ Live AI Prediction

GET `/predict-live?city=mumbai`

Output includes:

* real_temp
* predicted_temp
* humidity
* wind_speed
* clouds
* prediction_error
* confidence
